Provider API Reference

Connection Methods

  • eth_requestAccounts / hc_requestAccounts - Request account access
  • eth_accounts / hc_accounts - Get connected accounts

Network Methods

  • eth_chainId / hc_chainId - Get current chain ID
  • net_version / hc_networkVersion - Get network version

Signing Methods

  • personal_sign / hc_signMessage - Sign a message
  • eth_sendTransaction / hc_sendTransaction - Send transaction

Events

  • connect - Wallet connected
  • disconnect - Wallet disconnected
  • accountsChanged - Accounts changed
  • chainChanged - Network changed
